# AI Proctoring System for Online Exams

An AI-based online exam monitoring system that helps detect cheating during online examinations using webcam monitoring and face detection.

## 📌 Features

* Real-time webcam monitoring
* Face detection during exams
* Suspicious activity detection
* User-friendly exam interface
* Secure exam environment

## 🛠 Tech Stack

**Frontend**

* HTML
* CSS
* JavaScript

**Backend**

* Node.js
* Express.js

**AI / Detection**

* Face detection using computer vision techniques

## ⚙ Installation

1. Clone the repository

git clone https://github.com/sh1vamRajput/AI-Proctoring-System-Online-Exam.git

2. Go to project directory

cd AI-Proctoring-System-Online-Exam

3. Install dependencies

npm install

4. Start the server

npm start
